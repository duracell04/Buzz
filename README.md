# Buzz

Next.js (pages router) + Tailwind CSS + TypeScript, now set up to run with pnpm.

## Getting started

```sh
pnpm install
pnpm dev
```

## Scripts

- `pnpm dev` – start the Next.js dev server
- `pnpm build` – production build
- `pnpm start` – run the production build
- `pnpm lint` – lint with the current ESLint config

## Tech

- Next.js 14 + React 18
- Tailwind CSS + shadcn/ui components
- TypeScript
- React Query, Radix UI, Sonner for toasts
